## Packages
framer-motion | Smooth animations for page transitions and interactions
date-fns | Formatting dates for food expiration and listing times
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind classes safely

## Notes
- Replit Auth is used for user management.
- Integration uses `useAuth` hook.
- Images are handled via URL inputs (user provided), using Unsplash as placeholders.
- The app uses a "Green/Sustainable" aesthetic.
- "My Requests" page assumes the API returns requests the user has made.
